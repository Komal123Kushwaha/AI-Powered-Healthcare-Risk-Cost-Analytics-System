# 🏥 AI-Powered Healthcare Risk & Cost Analytics System

The **AI-Powered Healthcare Risk & Cost Analytics System** is a healthcare analytics project developed using **Python, SQL, Machine Learning, Power BI, and Streamlit**. The project focuses on analyzing patient healthcare data, identifying health risks, and generating meaningful insights to support better healthcare decision-making.

The system combines data analytics, visualization, and machine learning to transform raw healthcare records into actionable information. It helps healthcare professionals understand patient trends, monitor treatment costs, and identify patients who may require special medical attention.

## 🚀 Key Features

* 📊 Healthcare Data Analysis and Visualization
* 🤖 Patient Risk Prediction using Random Forest Classifier
* 💰 Treatment Cost Analysis
* 📈 Disease and Patient Trend Analysis
* 🌐 Interactive Streamlit Web Application
* 📋 Power BI Dashboard for Business Intelligence
* 🔍 Automated Healthcare Insights Generation

## 🤖 Machine Learning Model

A **Random Forest Classifier** is used to predict patient risk levels based on important healthcare parameters:

* Age
* BMI
* Blood Pressure (BP)
* Sugar Level
* Cholesterol

### Prediction Categories

* 🟢 Low Risk
* 🟡 Medium Risk
* 🔴 High Risk
