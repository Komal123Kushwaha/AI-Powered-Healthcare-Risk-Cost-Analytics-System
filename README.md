# 🏥 AI Powered Healthcare Risk & Cost Analytics System

An intelligent healthcare analytics system developed using SQL, Python, Machine Learning, and Power BI to analyze patient data, predict health risks, and visualize healthcare insights through interactive dashboards.

## Key Features

- Patient Data Analysis
- Disease Trend Analysis
- Treatment Cost Analysis
- Risk Prediction using Random Forest
- Interactive Power BI Dashboard
- Data Visualization using Python
- Streamlit-based User Interface

## Machine Learning Model

The project uses the Random Forest Classifier to predict patient risk levels based on healthcare parameters:

- Age
- BMI
- Blood Pressure (BP)
- Sugar Level
- Cholesterol

Output:
- Low Risk
- Medium Risk
- High Risk

## Technologies Used

- Python
- SQL (MySQL)
- Pandas
- NumPy
- Scikit-learn
- Random Forest Classifier
- Matplotlib
- Seaborn
- Power BI
- Streamlit

## Workflow

Healthcare Dataset
→ Data Cleaning
→ Exploratory Data Analysis
→ Risk Level Generation
→ Random Forest Model Training
→ Prediction
→ Dashboard Visualization
→ Streamlit Application

## Project Outcome

The system helps healthcare organizations identify high-risk patients, analyze treatment costs, and make data-driven decisions through predictive analytics and visualization.
